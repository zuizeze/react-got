# 使用react完成天天动听项目
## 项目架构
* 后端的接口采用的是网易云的后端接口，具体内容请查看[链接](https://github.com/Binaryify/NeteaseCloudMusicApi)
* 前端使用react+redux
## quickstart
1. 克隆这个项目
```
git clone git@github.com:zuizeze/react-got.git
```
```
cd react-got
```
2. 安装依赖
```
npm install
```
3. 启动这个项目
```
npm run start
```
4. 查看本地效果
	打开浏览器http://localhost:3000
